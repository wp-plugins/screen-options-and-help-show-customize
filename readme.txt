Customization options and help

== Description ==

You can set display options and getting help.
You can set the default page for almost all.
You can change the display for each user role.

== Installation ==

1. Upload the entire post-list-view-custom folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'Screen Options Customize' menu in your WordPress admin panel.
